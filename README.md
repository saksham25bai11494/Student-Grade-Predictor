# Student Grade Predictor

## 📌 Project Overview
This project predicts a student's **grade** based on two simple academic factors:

- Study Hours
- Attendance

The model predicts one of the following grades:

- A
- B
- C
- D

This is a simple beginner-friendly machine learning project.

---

## 🎯 Problem Statement
Students often want to understand how their study habits and attendance affect their academic performance.

This project solves that problem by predicting student grade using a machine learning model.
